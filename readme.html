TClock Light allows you to modify the clock on the taskbar in Windows.

TClock Lightは、タスクバーの時計を改造するプログラムです。

* How to use

Please refer to the following URL.
http://homepage1.nifty.com/kazubon/tclocklight/help/en/index.html

・使い方

次のURLを参照してください。
http://homepage1.nifty.com/kazubon/tclocklight/help/ja/index.html

* Compiling

You can compile and make TClock Light using Visual C++ or Borland C++ 
Compiler. The source codes are in "source" folder.

・コンパイル

TClock Lightは、Visual C++ または Borland C++ Compilerでコンパイルできます。
ソースコードはsourceフォルダの下にあります。
